# 🚗 AI Autonomous Navigation System

## 🧠 Overview

This project is an **AI-powered Autonomous Navigation System** that simulates intelligent path planning using the **A* (A-Star) algorithm** in a 2D grid environment.
The system allows an agent to navigate from a **Start (S)** position to a **Goal (G)** while avoiding obstacles and computing the most optimal path in real-time.
It demonstrates core concepts of:
* Artificial Intelligence
* Heuristic Search
* Path Optimization
* Obstacle Avoidance

---

## 🎯 Features

* 🧭 A* Pathfinding Algorithm for optimal route calculation
* 🚧 Dynamic obstacle handling
* 🎮 Real-time simulation using Pygame
* 📊 Grid-based environment visualization
* 🤖 Autonomous agent movement
* ⚡ Fast and efficient shortest-path computation

## 🎮 How It Works

1. A grid environment is generated with obstacles
2. The agent starts from position **S (Start)**
3. The goal is placed at position **G (Goal)**
4. The **A*** algorithm calculates the shortest path
5. The agent follows the path in real-time
